
Projet : Artisanry

Description :
Ce projet est un système web de gestion de produits artisanaux.

Technologies :
- HTML
- CSS
- PHP
- MySQL

Fonctionnalités :
1. Afficher les produits artisanaux.
2. Ajouter un nouveau produit.
3. Gestion simple des informations.

Installation :
1. Copier le dossier dans htdocs de XAMPP.
2. Démarrer Apache et MySQL.
3. Importer database.sql dans phpMyAdmin.
4. Ouvrir :
   http://localhost/artisanry_project/index.php

Explication :
- config.php : connexion base de données.
- index.php : page d'accueil.
- ajouter.php : ajout des produits.
- style.css : design du site.
- database.sql : création de la base de données.
